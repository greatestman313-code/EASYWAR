# CHAT EASY WAR

مشروع Next.js جاهز للنشر على Vercel — يحتوي الدردشة، المشاركة، التقارير، إعدادات الأدمن، تسجيل الدخول بجوجل، حراس حماية، وتكامل OpenAI.

## تشغيل محلي
```bash
npm i
cp .env.example .env   # عدّل القيم
npm run dev
```

## بناء ونشر
```bash
npm run build
npm run start
```

## اختبارات
- Unit: `npm run test:unit`
- E2E:  `npm run test:e2e`
